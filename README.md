# Break the Login

## 📌 Description
This project demonstrates how authentication systems can be attacked and how to secure them.

## ⚙️ Features
- Vulnerable login system
- Secure authentication with password hashing
- Attack simulation script

## 🛠 Technologies
- Python (Flask)
- HTML / CSS

## ▶️ How to run
1. Install dependencies
2. Run `init_db.py`
3. Run `app_vulner.py` or `app_secure.py`
